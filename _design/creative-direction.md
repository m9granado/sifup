# Direccion Creativa

## Idea central

SIFUP debe sentirse como una mezcla entre tablero deportivo, cultura de cancha amateur y cartel grafico de torneo barrial. No es una app corporativa fria ni una landing de marketing: es una herramienta viva para organizar partidos reales, cobrar pagos, mostrar rankings y mantener al grupo alineado.

La identidad toma energia del futbol global contemporaneo: color, diversidad, celebracion, movimiento y tipografias con caracter. La referencia es el espiritu multicolor y urbano de una gran fiesta futbolera, pero la marca debe ser propia de SIFUP. No debe usar logos, trofeos, slogans, mascotas, paises, sponsors ni recursos oficiales de FIFA o World Cup.

## Personalidad

- Futbolera
- Entretenida
- Competitiva sin ser agresiva
- De barrio, pero bien producida
- Rapida de leer en celular
- Con alma de grupo de WhatsApp llevado a producto

## Lenguaje visual

- Fondo oscuro tipo noche de cancha.
- Verdes de cancha y de UI actual como color base.
- Amarillos, cyan, magenta y rojos como acentos de energia.
- Letras pesadas, angulares, con sombra negra y textura.
- Lineas de cancha, trazos de velocidad, confetti y marcas tipo sticker.
- Componentes de producto claros: tablas, rankings, estados, deuda y resultados siempre legibles.

## Tipografia

Para UI, mantener una sans geometrica y legible. En la app actual usamos Geist, que funciona bien para tablas y navegacion.

Para piezas de marca, usar lettering custom o display pesado inspirado en rotulos deportivos: ancho, angular, con peso visual y textura. Evitar fuentes que se parezcan a identidades oficiales de torneos existentes.

## Reglas de uso

- La UI debe seguir siendo una herramienta operativa, no una pagina decorativa.
- En mobile, priorizar ranking, puntos, deuda y resultados por sobre decoracion.
- El key visual puede aparecer como fondo, pero siempre con overlays que aseguren contraste.
- El logo preferido actual tiene fondo y glow; usarlo como imagen/pieza, no como icono transparente.
- No usar marcas FIFA, World Cup, trofeo oficial, flags, sponsors ni slogans oficiales.
