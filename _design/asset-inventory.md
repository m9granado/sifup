# Inventario de Assets

## Logo preferido

Archivo: `assets/sifup-logo-preferred.png`

Uso recomendado:

- Presentaciones de marca.
- Header visual o splash.
- Piezas compartibles.
- Referencia principal para futuras versiones vectoriales.

Notas:

- Tiene fondo degradado, glow y textura.
- No es transparente.
- No conviene remover fondo de forma automatica porque se perderian bordes, luces y polvo negro.

## Logo preferido fuente

Archivo: `assets/sifup-logo-preferred-source.png`

Uso recomendado:

- Reprocesar recortes.
- Crear banners con mas aire.
- Referencia para una version manual transparente o vectorizada.

## Key visual

Archivo: `assets/sifup-keyvisual-v1.png`

Uso recomendado:

- Fondos de hero.
- Piezas visuales de campana.
- Imagen de referencia para el tono multicolor.
- Background en pantallas como Rankings con overlay oscuro.

Notas:

- Es una pieza completa con energia futbolera y colores diversos.
- No debe reemplazar UI code-native; solo debe apoyar fondos y atmosfera.

## Logo transparente exploratorio

Archivo: `assets/sifup-logo-transparent-v1.png`

Uso recomendado:

- Pruebas sobre fondos oscuros.
- UI cuando se necesite alpha real.
- Referencia secundaria, no version preferida.

Notas:

- Tiene bordes transparentes validados.
- Es mas limpio y menos crudo que el logo preferido.

## Logo con chroma

Archivo: `assets/sifup-logo-chroma-v1.png`

Uso recomendado:

- Fuente tecnica para regenerar transparencia.
- No usar en UI final.
