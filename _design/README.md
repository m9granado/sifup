# SIFUP Design Kit

Este directorio concentra la direccion creativa y los assets visuales que estamos construyendo para SIFUP.

SIFUP es una app para organizar partidos de futbol amateur: rankings, resultados, pagos, jugadores, equipos y coordinacion por WhatsApp. La identidad debe sentirse como futbol de barrio bien organizado: energia de martes en la noche, cancha, grupo, competencia sana y cobranza clara.

## Archivos principales

- `assets/sifup-logo-preferred.png`: logo preferido actual, recortado y listo para piezas visuales.
- `assets/sifup-logo-preferred-source.png`: fuente completa del logo preferido, con mas aire alrededor.
- `assets/sifup-keyvisual-v1.png`: key visual colorido para hero, fondos y piezas de comunicacion.
- `assets/sifup-logo-transparent-v1.png`: exploracion anterior del logo con fondo transparente.
- `assets/sifup-logo-chroma-v1.png`: fuente con chroma del logo transparente.

## Documentos

- `creative-direction.md`: descripcion de la creatividad, tono visual, paleta y reglas.
- `asset-inventory.md`: inventario de elementos disponibles y uso recomendado.
- `prompts.md`: prompts base usados para generar los assets y continuidad creativa.

## Estado actual

La version preferida del logo es `sifup-logo-preferred.png`. Mantiene textura, glow y fondo integrado; por eso no debe tratarse como logo transparente sin una pasada manual de recorte/limpieza. Para UI pequena o usos sobre fondos variables, usar por ahora la version transparente exploratoria o crear una version vector/alpha dedicada.
